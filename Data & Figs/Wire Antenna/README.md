# Thin-Wire Antenna Analytical Model ‚Äî Reproducibility Package

This folder contains a MATLAB live script implementing an analytical model for a thin-wire antenna based on characteristic modes (eigenchannels).
 The file is provided to support the **reproducibility and transparency** of the results reported in the accompanying manuscript under review.

### `Analytical_Model_WireAnt.mlx`

MATLAB live script implementing:

- Analytical expressions for **characteristic-mode (eigenchannel) currents** of a thin-wire antenna
- Computation of **far-field radiation patterns** for individual eigenchannels
- Demonstration of **current asymmetry** induced by superposition of multiple eigenchannels
- Visualization of **far-field intensity** using a 3D cut-shell representation

The script includes:

- Parameter definitions
- Analytical model equations
- Visualization routines
- Fully reproducible figure generation

All figures can be regenerated by running the live script.

------

## ‚ñ∂ How to Run

1. Open MATLAB.

2. Set the current folder to this directory.

3. Open the file:

   ```
   Analytical_Model_WireAnt.mlx
   ```

4. Click **Run All** in the MATLAB Live Editor.

All figures will be generated automatically.
